# Чат Анализаторы бойынша есеп

## Мақсаты:
HashMap коллекциясын қолдану арқылы чат деректерін өңдеу және статистика жасау.

## Сұрақтарға жауап:
1. **HashMap қалай жұмыс істейді?**
   HashMap — бұл деректерді "кілт-мән" (key-value) жұбымен сақтайтын құрылым. Ол хэштеу механизмін қолданады, сондықтан мәліметті іздеу өте жылдам.

2. **Жиілік қалай есептеледі?**
   Сөздердің немесе хабарламалардың жиілігін есептеу үшін цикл ішінде `map.put(key, map.getOrDefault(key, 0) + 1)` әдісі қолданылады. Егер кілт бар болса, мәні артады, жоқ болса — 1 болып қосылады.

3. **Қолданушыларды қалай анықтауға болады?**
   Хабарлама жолын `split(": ")` әдісімен бөлу арқылы бірінші элементті (индекс 0) қолданушының есімі ретінде анықтаймыз.
